 Personal Portfolio Website

 Description
This is a Personal Portfolio Website designed to showcase my skills and experience. The website features a product gallery that displays various items in a clean and organized layout. It is fully responsive and works seamlessly across different devices, providing a professional overview of my work and skills.

Technologies Used
- HTML: Used for the structure of the website.
- CSS: Applied to style the layout, ensuring the site is responsive and visually appealing.
- JavaScript: Added for interactivity and dynamic features on the page.
- GitHub Pages: Used to host the project and make it publicly accessible.

 Features
- Responsive Design: The website adjusts to different screen sizes, ensuring a consistent experience across mobile and desktop devices.
- Product Gallery: A section displaying various products. 
- Live Demo: The website is hosted on GitHub Pages and is accessible online.

Installation/Setup Instructions
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/matthewbarghout/website.git
   ```
2. Navigate into the project directory:
   ```bash
   cd website
   ```
3. Open the `index.html` file in your browser:
   ```bash
   open index.html
   ```
   
Live Demo
You can view the live version of the website [here](https://matthewbarghout.github.io/website/).
